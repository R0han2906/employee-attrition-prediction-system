# 🧠 IBM HR Attrition Prediction System

An end-to-end Machine Learning application that predicts employee attrition risk and explains predictions using SHAP interpretability.  

Built with FastAPI backend and a clean HTML/CSS/JS frontend.

## 🧠 Model Overview

- Algorithm: Elastic Net Logistic Regression  
- Metric Used: ROC-AUC + Recall Optimization  
- Threshold Tuned for Business Objective  
- Top Drivers Identified via SHAP:
  - Monthly Income
  - Overtime
  - Job Satisfaction
  - Years Since Last Promotion
  - Distance From Home
